Problema:

      Data un interfaccia IShape per una generica forma geometrica:
      1. definire una classe Rectangle che implementi IShape
      2. definire una classe Square che estenda Rectangle
      3. come si puo' implementare una funzione che calcoli l'area totale dato un IShape[]?

      
Implementazione (punti 1 e 2):

      Inserire le classi nel package it.ott8bre.sandbox.models

      
Unit Test (punto 3):

      Utilizzare il metodo shapesTotalAreaTest della classe it.ott8bre.sandbox.SandboxApplicationTests
